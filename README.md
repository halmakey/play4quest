# Play4Quest

Play4Quest converts YouTube video URLs so they can be played on Quest as well.   
(It performs the conversion known as Jinnaize.)

## Environment

- Node.js 18
- macOS or Linux
- AWS CDK

## Setup

```sh
npm i
```

## Run (Local test)

```sh
cdk synth && npm start
```

Open http://127.0.0.1:3000

## Deploy

```sh
cdk deploy
```

## Usage

Open https://play4.quest/
